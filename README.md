# Тестове завдання ML Engineer Trainee: Детекція транспортних засобів YOLOv8

Тренування YOLOv8 nano на датасеті Vehicle Detection (Roboflow).

## Основні метрики (validation)
- Precision: 0.816
- Recall: 0.789
- mAP@50: 0.835
- mAP@50:95: 0.459

## Структура
- `vehicle_detection_yolov8.ipynb` — повний notebook з аналізом, тренуванням, оцінкою та прикладами
- `examples/` — приклади передбачень (вдалі та невдалі)
